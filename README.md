## Assembly 
### Display
RST -> D3  
CS -> D8  
DIN -> D7  
CLK -> D5  
VCC -> VCC  
BL -> D6  
GND -> GND  
### Buttons
K1 -> D4  
K2 -> D2  
K3 -> D1  
K4 -> RST  
GND -> GND  
### Battery
B- -> Batt-  
B+ -> Batt+  
OUT- -> GND  
OUT+ -> VCC
